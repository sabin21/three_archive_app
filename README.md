three.js- Archive
========

#### WebGL(Theree.js) 예제,자료 링크 모음 ####

WebGL, Three.js를 활용한 간단한 기본교육용 예제 파일, 디자인작업에 활용될 수 있는 예제 파일등을
아카이브하는 목적으로 제작됨


### Usage ###

1. Node.js 설치

2. yarn 설치

3. package 설치

4. Webpack Dev server 실행

```javascript
yarn run start
```

5. Webpack Build 
```javascript
yarn run build
```
